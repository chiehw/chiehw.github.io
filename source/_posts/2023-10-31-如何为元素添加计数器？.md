---
title: 如何为元素添加计数器？
date: 2023-10-31 22:27:40
categories: 程序人生
tags:
---

- 在适当的位置初始化计数器

```css
counter-reset: drag; 
```

- 为需要计数的元素设定 css

```css
.is-question {
  counter-increment: drag 1;
}
```

- 将计数器的文字展示出来。

```css
.is-question:before {
  content: counter(drag, decimal-leading-zero);
  padding-left: 20px;
}
```
